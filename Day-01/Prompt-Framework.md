# Day 1: The AI Handshake Protocol

### Objective: Establishing Context
The "Handshake" starts with clear intent. Every prompt should include:
1. **Role:** Who is the AI? (e.g., Expert Technician)
2. **Task:** What is it doing? (e.g., Troubleshooting a JCB engine)
3. **Format:** How should it answer? (e.g., Bulleted list)
